# Glossary

* **Referee.** The person who narrates the consequences of protagonist actions and adjudicates the rules.  
* **Player.** Anyone playing the game besides the referee.  
* **Protagonist.** A character controlled by a player.  
* **Non-protagonist Character (NPC).** A character controlled by the referee.  
* **Invoke.** When a game element (such as a Trait) is activated to produce an effect  
* **Compel.** When a game element can potentially create difficulty for a protagonist, that element can be compelled to cause that difficulty in exchange for some benefit.  
* **Upgrade.** When a value is upgraded, it is increased to the specified value unless it is already above that value.  
* **Refresh.** If a protagonistâ€™s resource is refreshed, they regain any of that resource that they spent. If they have more of that resource than their refresh value, they lose the excess.  
* **Regain.** A protagonist can often regain a limited amount of a resource. The amount of a resource gained this way cannot make the protagonist exceed their refresh value in that resource. Any excess is lost.
