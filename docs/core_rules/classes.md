# Classes

Classes define a protagonist’s archetype and long-term thematic direction. A protagonist’s Class is chosen when they are created.

## Core Class Features

When a protagonist gains their Class, they gain all of the Core Class Features for their chosen Class.

## Class Ranks

A protagonist has 1 to 10 ranks in their Class.

## Major Class Features

At each Major Milestone, a protagonist gains a rank in their Class and gains the Major Class Features for that rank in their Class.
